# Business Website

A fully functional business website currently deployed and running in production.

## Live Website

🔗 https://www.servicentermx.com

## Features

* Multi-page structure (Home, Services, About, Contact)
* Responsive design for mobile and desktop
* Interactive navigation menu
* Integrated map and dynamic elements

## Technologies

* HTML5
* CSS3
* JavaScript

## Project Structure

* /css → styles
* /js → functionality
* /img → images
* /videos → media

## Description

This project consists of a real-world business website developed and deployed with a custom domain.
It was designed to provide information about services, improve online presence, and offer a professional user experience.
